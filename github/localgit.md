# ローカルPCにgitをインストールする。

WindowsにGitをインストールする手順。

0. chocolateyをインストール  
https://chocolatey.org/install  
[chocolateyに関して](https://github.com/snjxnksm/javaenv/blob/master/steps/chocolatey_install.md)
0. gitをインストール
管理者権限でpwershellを開いて...　　
```
> choco install git -y
```
